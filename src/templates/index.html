Setting Up

To play PTD3 using a personal server, you must:

  1. Create a Deta Account and install the Server on it (If you are here, this step is already done)
  2. Create an API Key for the App on your Space Canvas
    1. Go the Space Canvas
    2. Find the PTD3-SERVER App
    3. Click the (...) and open Settings
    4. Go to the Keys tab and scroll down to API Keys
    5. Create an API Key. Name it whatever you want ; Make sure to save it somewhere.
  3. Download the modified SWF for PTD3 and a Flash Player if you do not have one yet. If the above links do not work, please check if there is more information (e.g. updates or new links) mentioned in the GitHub page. If you are a programmer and wish to make your own 'fork' of the client, you can download the instructions to modify it yourself here.
  4. Once you have the server, a flash player and the SWF file, open the SWF file or load it from the player. After the game opens, you'll have to use the following as your login:
    • Server URL: The link to this server (https://something.deta.app/), with /ptd3save/username as the path
      • The url to this server should start with something like [ptd3server-small number-random sequence of characters]. Make sure you copy paste yours!
      In the end, it should look something like https://ptd3server-1-abc123xyz.deta.app/ptd3save/shiro
    • Server Password: The API Key you created previously
